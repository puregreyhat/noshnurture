  User clicks "Translate" button
    ↓
  Sarvam AI translates to Indian language (हिंदी, मराठी, etc.)
    ↓ (if Sarvam AI available & has API key)
  Browser API tries (instant if Chrome 140+)
    ↓ (if unavailable)

    ↓ (if unavailable)
  English shown (always readable)
    ✓ User happy either way!

🔑 SETUP REQUIRED (3 STEPS):

  1. Get API Key:
     Visit: https://console.sarvam.ai
     Sign up (free) → Create API key → Copy

  2. Add to .env.local:
     NEXT_PUBLIC_SARVAM_API_KEY=your_key_here

  3. Test:
     npm run dev
     → Select recipe → Change to हिंदी → Click Translate ✨
